    Installation #

    Es gibt verschiende Möglichkeiten MapProxy zu installieren:

    • Systemweite Installation (wie auf der OS Geo Live)
      python -m pip install MapProxy
    • Virtual Environment (Anleitung)
    • Über ein docker image (z.B. kartoza)
